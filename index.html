About Me

I am Kitso Senwedi, an IT professional specializing in software development and cloud solutions. I hold a Bachelor's degree in Information Technology, majoring in Software Development. My expertise includes web development, React.js, AWS cloud architecture, and blockchain technologies.

My experience includes working as a freelance frontend developer, where I built and hosted a static website for a client. Throughout my IT degree, I completed modules such as Software Engineering, Web Development, Database Systems, Cybersecurity, Cloud Computing, Artificial Intelligence, and Blockchain Technologies.
